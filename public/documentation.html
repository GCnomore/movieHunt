Documentation

Movie Hunt API will deliver various information about a movie. (Description, genre, postere etc.) It will also allow users to register & deregister. Registered users will be able to add movies to their list of favorites.

Request URL HTTP Method Request body data format Response body data format
Return all movies
(20 movies per page by default) 		/movies GET Array of JSON object
{
    movies :
       [{
          title : "Inception",
          genre : "Sci-fi",
          director : "Christopher Nolan",
          id : "23psdojf"
       }]
}
Return all data of a movie by title /movies/data/:title GET Array of JSON object
{
    movieData :
       [{
          title : "Inception",
          description : "Dom Cobb (Leonardo DiCaprio) is a thief...",
          genre : "Sci-fi",
          director : "Christopher Nolan",
          imgURL : "https://...jpg",
          featured : "true" || "false",
          id : "23psdojf"
       }]
}
Return a genre's description by genre name /genres/:genre GET JSON object
{
       name : "Sci-fi",
       description : "Science fiction (or sci-fi) is a film genre that uses speculative..."
}
Return a director's data by name /directors/:directorName GET JSON object
{
       director : "Christopher Nolan",
       dob : "July 30, 1970",
       dod : null,
       bio : "Best known for his cerebral, often nonlinear, storytelling, acclaimed writer-director........"
}
New user register /users/new POST JSON object
*User name has to be all lower case w/o spaces
*Password should be at least 6 characters

{
       username : "New_user_name",
       password : "******",
       email : 123@456.com,
       dob : "MM/DD/YYYY"
} 		Confirmation message


*On success
    "New user {username} has been registered!"

*On error
    "Error occurred. Please try again."
    + error code
Update user info /users/:username PUT JSON object
*User name has to be all lower case w/o spaces
*Password should be at least 6 characters

{
       username : "user_name",
       password : "******",
       email : 123@456.com,
       dob : "MM/DD/YYYY"
} 		Confirmation message


*On success
    "{username} has been updated!"

*On error
    "Error occurred. Please try again."
    + error code
Add a movie to favorite list
(Favorite list only available for registered users) 		/users/:username/favorite/:movieID PUT Confirmation message


*On success
    "{movieName} has been added to {username}'s favorite list!"

*On error
    "Error occurred. Please try again."
    + error code
Remove a movie from favorite list /users/:username/favorite/:movieID DELTE Confirmation message


*On success
    "{movieName} has been removed from {username}'s favorite list!"

*On error
    "Error occurred. Please try again."
    + error code
Deregister existing user /users/:username DELTE Confirmation message


*On success
    "{username} has been deregisterd"

*On error
    "Error occurred. Please try again."
    + error code
